Dam Reservoir Monitoring System

The idea is to read Level data from already installed sensors and convey it to a web based application that can indicate these parameters on a conventional mobile phone or a remote PC that has internet connectivity. However, the application will be using the GSM/public domain and AR cannot guarantee the integrity of data.

In addition to the RTU and the hardware (we have developed the RTUs), it will require a webserver hosting application. The data can be retrieved by the user once he logs in using his ID and password. A small Android software application can be developed to enable this in a mobile phone

  1. Two transcievers to establish connectivity between RTUs
  2. One remote Slave RTU
  3. One Master RTU as data concentrator (to be developed)
  4. One WiFi/GSM dongle and SIM card
  5. Two Power supplies 230VAC to 12VDC/1Amp for the RTUs.

This product will be commercially available very soon.
