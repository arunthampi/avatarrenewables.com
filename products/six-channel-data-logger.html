Six Channel Data Logger

This a low-cost stand-alone Data Logger for industrial applications. It can accept 6 analog inputs that can be either 4-20mA or 1-5 V industrial standard.

The built-in backlit LCD displays all values in engineering units. The 10-bit raw values for the inputs are stored with timestamp (DD/MM/20YY HH:MM:SS) in a micro SD card in CSV file format. Typically an 8-GB card can store 1 year's data sampled at 1 second (roughly 2 million samples).

This format helps in manipulating the data later using Excel or similar spreadsheet applications. The user can set the ranges of each input as well as select the SD-card recording frequency. Push-buttons are provided to configure date/time and ranges as well as to start and stop recording.
