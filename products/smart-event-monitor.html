Smart Event Monitor

This is a standalone product that can perform as a local indicator for various package equipment. It can detect the “first-in” status signal which is an important feature that is not available in many competing indicators.

The first-in input LED will keep flashing until it becomes normal. Once it is normalized, the second in the sequence keeps flashing and so on. This becomes advantageous in identifying the input signal that could have caused an equipment trip. The input statuses, number of events and the “first-in” can be remotely monitored through an RS485 communication link.

This product will be commercially available very soon.
